# Instalación
---

**Requisitos**
- Navegador web Moderno (Chrome, Safari, Edge, Firefox)
- Conexión a internet estable
- Cuenta de usuario en moonFit (gratuita o premium)

**Pasos de instalación**
1. Acceder a la plataforma moonFit desde el navegador.
2. Hacer clic en "Registrarse" y crear una cuenta.
3. Confirmar la cuenta mediante el correo electrónico.
4. Iniciar sesión y configurar perfil de usuario (edad, peso, objetivos de entrenamiento).

**Variables de entorno**
- `DB_HOST` → Dirección del servidor de la base de datos (ejemplo: localhost)
- `DB_USER` → Usuario para acceder a la base de datos
- `DB_PASSWORD` → Contraseña del usuario de la base de datos
- `DB_NAME` → Nombre de la base de datos de MoonFit
- `API_KEY` → Clave de acceso a servicios externos (opcional)
